# Simulateur Retraite

Application statique React standalone, compatible GitHub Pages.

## Déploiement GitHub Pages gratuit
1. Créer un repository GitHub public ou privé.
2. Uploader tous les fichiers de ce dossier à la racine du repository.
3. Aller dans Settings > Pages.
4. Source : Deploy from a branch.
5. Branch : main / root.
6. Ouvrir l'URL GitHub Pages générée.

## Calcul pension
La pension mensuelle est calculée en consommation progressive du capital jusqu'à 85 ans :
- âge actuel paramétrable dans l'onglet Horizon ;
- durée de décaissement = 85 ans - âge à l'année de retraite ;
- rendement mensuel pendant la phase de retrait ;
- formule d'annuité financière.


## Version mobile complète
Cette version garde tous les onglets et calculs, avec adaptation mobile iPhone.
